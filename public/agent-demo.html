The Multi-Agent System

FinDoc Analyzer uses a multi-agent system to provide comprehensive analysis of financial documents. Each agent specializes in a specific aspect of financial document analysis.

Document Analyzer Agent

The Document Analyzer agent is responsible for understanding the general content of the document.

Model: Claude 3 Sonnet (via OpenRouter)

Specialization: General document understanding and basic information extraction

Example Interaction

Q: What is the total value of the Messos portfolio?

A: Based on the document data, the total value of the Messos portfolio is $19,510,599 as of February 28, 2025.

Table Understanding Agent

The Table Understanding agent specializes in extracting and analyzing tabular data from documents.

Model: Claude 3 Sonnet (via OpenRouter)

Specialization: Parsing and analyzing tables, extracting structured data

Example Interaction

Q: What is the asset allocation of the Messos portfolio?

A: The asset allocation of the Messos portfolio is:
- Bonds: $11,558,957 (59.24%)
- Structured Products: $7,851,642 (40.24%)
- Cash: $100,000 (0.52%)

Securities Extractor Agent

The Securities Extractor agent focuses on identifying and extracting information about financial securities.

Model: DeepSeek Coder (via OpenRouter)

Specialization: Identifying securities, extracting details, categorizing instruments

Example Interaction

Q: List all the bond holdings in the Messos portfolio.

A: The bond holdings in the Messos portfolio are:
1. US Treasury Bond 2.5% 2030: $2,500,000 (12.81%)
2. German Bund 1.75% 2032: $2,000,000 (10.25%)
3. JP Morgan Corporate Bond 3.25% 2028: $1,200,000 (6.15%)

Financial Reasoner Agent

The Financial Reasoner agent performs financial analysis and calculations based on the document data.

Model: Claude 3 Sonnet (via OpenRouter)

Specialization: Financial analysis, calculations, ratios, performance metrics

Example Interaction

Q: What is the risk profile of the Messos portfolio?

A: Based on the asset allocation, the Messos portfolio has a moderate risk profile. With 59.24% in bonds, 40.24% in structured products, and 0.52% in cash, it balances stability (bonds and cash) with potential higher returns (structured products). The significant allocation to structured products (40.24%) introduces some complexity and potentially higher risk compared to a traditional bond-heavy portfolio.

Bloomberg Agent

The Bloomberg Agent fetches real-time financial data from external sources to provide context and current information.

Model: Gemini Pro (via OpenRouter)

Specialization: Real-time market data, current prices, market trends

Example Interaction

Q: What is the current price of the US Treasury Bond 2.5% 2030?

A: As the Bloomberg Agent, I would normally fetch the current price of the US Treasury Bond 2.5% 2030 from financial data providers. Based on recent market data, this bond is trading at approximately 98.75, yielding about 2.65%. Please note that this is an estimate and the actual current price may vary based on market conditions.

How the Agents Work Together

The power of this multi-agent system comes from how they collaborate to provide comprehensive analysis:

The Document Analyzer first understands the document structure and extracts basic information
The Table Understanding agent extracts detailed tabular data
The Securities Extractor identifies specific securities and their details
The Financial Reasoner performs calculations and analysis
The Bloomberg Agent adds real-time market context

Example of Collaborative Analysis

When a user asks "What is the current value of my US Treasury Bond compared to its valuation in the document?", the system:

Routes the question to the Bloomberg Agent (due to "current value")
Accesses the document data (from Document Analyzer)
Identifies the US Treasury Bond (using Securities Extractor's categorization)
Retrieves the valuation from the document ($2,500,000)
Fetches the current market price from external sources
Calculates the difference and provides a comprehensive answer
