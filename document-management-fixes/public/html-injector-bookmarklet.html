HTML Injector Bookmarklet

This bookmarklet injects the UI components into the FinDoc Analyzer application.

How to use:

  1. Drag the bookmarklet below to your bookmarks bar.
  2. Navigate to any page on the FinDoc Analyzer application.
  3. Click the bookmarklet to inject the UI components.

Drag this link to your bookmarks bar:

Inject UI Components

What this bookmarklet does:

  • Injects the UI components script into the page
  • Adds the process button to the upload page
  • Adds the chat button to all pages
  • Adds the agent cards to the test page

Testing the UI components:

You can test the UI components by visiting the UI Components Test page.
